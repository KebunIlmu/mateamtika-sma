1. Pengertian Polinomial

Polinomial adalah bentuk aljabar yang merupakan penjumlahan atau pengurangan dari suku-suku yang terdiri atas koefisien dikalikan dengan variabel berpangkat bilangan bulat non-negatif. Bentuk umumnya dapat ditulis:

P(x) = anxn + an-1xn-1 + ... + a1x + a0
  • P(x) : nama polinomial
  • an, an-1, ..., a0 : koefisien (bilangan real)
  • x : variabel
  • n : derajat polinomial (bilangan bulat non-negatif)
  • Suku : setiap bagian polinomial yang dipisahkan oleh tanda + atau âˆ’

Contoh Polinomial

2x + 3,   x2 âˆ’ 5x + 6,   4x3 + 2x2 âˆ’ x + 7

Bukan Polinomial

  • 3x-1 + 2 (pangkat negatif)
  • âˆšx + 1 (pangkat pecahan)
  • 5/x + 2 (variabel di penyebut)
  • 2x + 3 (variabel sebagai pangkat)

5. Teorema Faktor

Jika P(c) = 0, maka (x âˆ’ c) adalah faktor dari P(x). Artinya, bila suatu nilai c membuat polinomial sama dengan nol, maka (x âˆ’ c) membagi habis P(x).

Contoh Soal:

Tentukan apakah (x - 2) merupakan faktor dari polinomial P(x) = xÂ³ - 4xÂ² + 5x - 2.

Pembahasan:

Untuk menentukan apakah (x - 2) adalah faktor dari P(x), kita substitusi c = 2 ke dalam P(x):

P(2) = 2Â³ - 4(2)Â² + 5(2) - 2 = 8 - 16 + 10 - 2 = 0

Karena P(2) = 0, maka (x - 2) adalah faktor dari P(x).

6. Teorema Sisa

Sisa pembagian P(x) oleh (x âˆ’ c) sama dengan P(c). Jadi, untuk mengetahui sisa pembagian, cukup evaluasi polinomial P pada x = c.

Contoh Soal:

Diketahui polinomial f(x) memiliki sifat berikut:

Tentukan sisa pembagian f(x) oleh (xÂ² - 4).

Pembahasan:

\[ R(x) = ax + b \]

Dengan menggunakan teorema sisa, kita substitusi nilai \(x = 2\) dan \(x = -2\):

  • \(R(2) = 2a + b = f(2) = 4\)

\[ \begin{cases} 2a + b = 4 \\ -2a + b = -8 \end{cases} \]

Langkah 2: Cari nilai \(a\) dan \(b\)

\[ 2(3) + b = 4 \implies 6 + b = 4 \implies b = 4 - 6 = -2 \]

Kesimpulan:

Jadi, sisa pembagian f(x) oleh \((x^2 - 4)\)adalah:

\[ R(x) = 3x - 2 \]

7. Metode Partial

Metode Partial digunakan untuk memecah fungsi rasional menjadi pecahan sederhana (partial fractions), berguna dalam integral dan analisis aljabar.

Contoh Soal:

\[ \frac{2x + 3}{x^2 - 1} = \frac{A}{x - 1} + \frac{B}{x + 1} \]

Cari nilai A dan B sehingga persamaan tersebut benar.

Langkah 1: Faktorkan penyebut

\( x^2 - 1 = (x - 1)(x + 1) \), sehingga kita pisahkan menjadi dua pecahan sederhana.

Langkah 2: Kalikan kedua ruas dengan penyebut bersama

\[ 2x + 3 = A(x + 1) + B(x - 1) \]

Langkah 3: Cari nilai A dan B dengan substitusi nilai x

  • Untuk x = 1:

    2(1) + 3 = A(1 + 1) + B(0) â†’ 5 = 2A â†’ A = 2.5

  • Untuk x = -1:

    2(-1) + 3 = A(0) + B(-1 - 1) â†’ 1 = -2B â†’ B = -0.5

Kesimpulan:

Jadi, \[ \frac{2x + 3}{x^2 - 1} = \frac{2.5}{x - 1} - \frac{0.5}{x + 1} \]

7b. Metode Partial: Penyebut (x - 1)(x^2 + 2)

Jika penyebut merupakan hasil perkalian faktor linear dan kuadrat irreducible, bentuk partial fraction adalah kombinasi pecahan dengan penyebut masing-masing faktor.

Contoh Soal:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x^2 + 2)} = \frac{A}{x - 1} + \frac{Bx + C}{x^2 + 2} \]

Tentukan nilai A, B, dan C.

Langkah 1: Bentuk partial fraction

Bentuk pecahan parsial adalah: \[ \frac{A}{x - 1} + \frac{Bx + C}{x^2 + 2} \]

Langkah 2: Kalikan kedua ruas dengan penyebut

\[ 3x^2 + 5x + 2 = A(x^2 + 2) + (Bx + C)(x - 1) \]

Langkah 3: Kembangkan ruas kanan

\[ 3x^2 + 5x + 2 = A x^2 + 2A + B x^2 - B x + C x - C \]

Langkah 4: Gabungkan suku sejenis

\[ 3x^2 + 5x + 2 = (A + B) x^2 + (-B + C) x + (2A - C) \]

Langkah 5: Samakan koefisien

  • Koefisien \(x^2\): \(3 = A + B\)
  • Koefisien \(x\): \(5 = -B + C\)
  • Konstanta: \(2 = 2A - C\)

Langkah 6: Selesaikan sistem persamaan

Dari persamaan di atas:

\[ \begin{cases} 3 = A + B \\ 5 = -B + C \\ 2 = 2A - C \end{cases} \]

Dengan substitusi:

  • \(B = 3 - A\)
  • \(5 = -(3 - A) + C \Rightarrow C = 8 - A\)

  • \(2 = 2A - (8 - A) \Rightarrow 3A = 10 \Rightarrow A = \frac{10}{3}\)

  • \(B = 3 - \frac{10}{3} = -\frac{1}{3}\)
  • \(C = 8 - \frac{10}{3} = \frac{14}{3}\)

Kesimpulan:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x^2 + 2)} = \frac{\frac{10}{3}}{x - 1} + \frac{-\frac{1}{3}x + \frac{14}{3}}{x^2 + 2} \]

7c. Metode Partial: Penyebut dengan Pangkat

Jika penyebut mengandung faktor pangkat, misal \((x - 1)(x + 1)^2\), bentuk partial fractions harus mencakup pecahan untuk setiap pangkat tersebut.

Contoh Soal:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x + 1)^2} = \frac{A}{x - 1} + \frac{B}{x + 1} + \frac{C}{(x + 1)^2} \]

Tentukan nilai A, B, dan C.

Langkah 1: Bentuk partial fraction lengkap

Karena ada faktor \((x+1)^2\), kita buat dua pecahan: satu untuk pangkat 1 dan satu untuk pangkat 2.

Langkah 2: Kalikan kedua ruas dengan penyebut

\[ 3x^2 + 5x + 2 = A(x + 1)^2 + B(x - 1)(x + 1) + C(x - 1) \]

Langkah 3: Substitusi nilai x untuk cari A, B, C

  • Untuk x = 1:
    \(3(1)^2 + 5(1) + 2 = A(1 + 1)^2 + B(0) + C(0)\) â†’ \(3 + 5 + 2 = A \times 4\) â†’ \(10 = 4A\) â†’ \(A = \frac{10}{4} = 2.5\)
  • Untuk x = -1:
    \(3(-1)^2 + 5(-1) + 2 = A(0) + B(-1 - 1)(0) + C(-1 - 1)\) â†’ \(3 - 5 + 2 = C \times (-2)\) â†’ \(0 = -2C\) â†’ \(C = 0\)

Langkah 4: Cari B dengan substitusi nilai x lain

Pilih nilai lain, misal x = 0:

\(3(0)^2 + 5(0) + 2 = A(0 + 1)^2 + B(0 - 1)(0 + 1) + C(0 - 1)\)
\(2 = A(1)^2 + B(-1)(1) + C(-1)\)
\(2 = A - B - C\)

Dengan \(A = 2.5\) dan \(C = 0\), maka:

\[ 2 = 2.5 - B - 0 \implies B = 2.5 - 2 = 0.5 \]

Kesimpulan:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x + 1)^2} = \frac{2.5}{x - 1} + \frac{0.5}{x + 1} + \frac{0}{(x + 1)^2} = \frac{2.5}{x - 1} + \frac{0.5}{x + 1} \]

Karena \(C = 0\), suku terakhir hilang.
