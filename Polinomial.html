1. Pengertian Polinomial

Polinomial adalah bentuk aljabar yang merupakan penjumlahan atau pengurangan dari suku-suku yang terdiri atas koefisien dikalikan dengan variabel berpangkat bilangan bulat non-negatif. Bentuk umumnya dapat ditulis:

P(x) = anxn + an-1xn-1 + ... + a1x + a0
  • P(x) : nama polinomial
  • an, an-1, ..., a0 : koefisien (bilangan real)
  • x : variabel
  • n : derajat polinomial (bilangan bulat non-negatif)
  • Suku : setiap bagian polinomial yang dipisahkan oleh tanda + atau −

Contoh Polinomial

2x + 3,   x2 − 5x + 6,   4x3 + 2x2 − x + 7

Bukan Polinomial

  • 3x-1 + 2 (pangkat negatif)
  • √x + 1 (pangkat pecahan)
  • 5/x + 2 (variabel di penyebut)
  • 2x + 3 (variabel sebagai pangkat)

📘 Soal Matematika - Suku Banyak

Sebutkan variabel, derajat, dan koefisien tiap suku banyak berikut:

a. 5x³ - 2x² + 10x + 4
b. y⁴ + 4y³ - 3y² + y + 2
c. (x - 2)²
d. (x + 2)(x - 4)
e. (t - 4)²(t + 1)
f. (y - 2)(2y + 1)(4y + 1)²

2. Operasi Polinomial

Operasi dasar yang biasa dipelajari pada polinomial meliputi:

Penjumlahan & Pengurangan

Susun suku sejenis (pangkat sama) lalu jumlahkan atau kurangkan koefisiennya.

(3x2 + 2x + 5) + (4x2 − x + 3) = (3 + 4)x2 + (2 − 1)x + (5 + 3) = 7x2 + x + 8
(5x3 + 4x − 2) − (2x3 − x + 7) = (5 − 2)x3 + (4 + 1)x + (−2 − 7) = 3x3 + 5x − 9

Perkalian

Kalikan setiap suku pada polinomial pertama dengan setiap suku pada polinomial kedua, kemudian gabungkan suku sejenis.

(x + 2)(x + 3) = x(x + 3) + 2(x + 3) = x2 + 3x + 2x + 6 = x2 + 5x + 6

Pembagian (Ringkasan)

Pembagian polinomial dapat dilakukan dengan metode pembagian bersusun atau Horner. Rumus khusus tidak sederhana, sehingga akan dijelaskan pada bagian tersendiri.

Faktorisasi

Mencari faktor-faktor polinomial, misalnya dengan mencari akar (nilai x yang membuat polinomial = 0) lalu menuliskan dalam bentuk perkalian faktor linear.

x2 + 5x + 6 = (x + 2)(x + 3)
1. Hasil dari (3x² + 2x + 5) + (4x² - 3x + 1) adalah ...
(3x² + 2x + 5) + (4x² - 3x + 1) = (3x² + 4x²) + (2x - 3x) + (5 + 1) = 7x² - x + 6 → **Jawaban: A**
2. Hasil dari (5x² - 3x + 7) - (2x² + 4x - 5) adalah ...
(5x² - 3x + 7) - (2x² + 4x - 5) = (5x² - 2x²) + (-3x - 4x) + (7 + 5) = 3x² - 7x + 12 → **Jawaban: A**
3. Hasil dari (x + 3)(x - 2) adalah ...
(x + 3)(x - 2) = x² - 2x + 3x - 6 = x² + x - 6 → **Jawaban: A**
4. Hasil dari (x + 2)(x² - 3x + 4) adalah ...
(x + 2)(x² - 3x + 4) = x³ - 3x² + 4x + 2x² - 6x + 8 = x³ - x² - 2x + 8 → **Jawaban: A**
5. Hasil dari (2x² - 3x + 1) + (x² + 4x - 5) - (3x² - 2x + 7) adalah ...
(2x² - 3x + 1) + (x² + 4x - 5) - (3x² - 2x + 7) = 2x² + x² - 3x² + (-3x + 4x + 2x) + (1 - 5 - 7) = -x² + 3x - 11 → **Jawaban: A**
6. Hasil dari (x - 4)(x² + 2x - 1) adalah ...
(x - 4)(x² + 2x - 1) = x³ + 2x² - x - 4x² - 8x + 4 = x³ - 2x² - 9x + 4 → **Jawaban: A**
7. Hasil dari (2x + 3)(x² - x + 5) - (x³ - 2x² + 4) adalah ...
(2x + 3)(x² - x + 5) = 2x³ - 2x² + 10x + 3x² - 3x + 15 = 2x³ + x² + 7x + 15 Lalu dikurangi (x³ - 2x² + 4): (2x³ - x³) + (x² + 2x²) + 7x + (15 - 4) = x³ + 3x² + 7x + 11 → **Jawaban: A**

3. Nilai Polinomial

Untuk mendapatkan nilai polinomial pada \(x = c\), substitusikan nilai \(c\) ke \(P(x)\) dan hitung.

Contoh: \[ P(x) = x^2 - 3x + 2 \] Maka \[ P(2) = 2^2 - 3 \cdot 2 + 2 = 4 - 6 + 2 = 0 \]

Metode Horner juga mempercepat evaluasi nilai \(P(c)\) (lihat bagian 4b.i).

1. Jika f(x) = 2x² + 3x - 5, maka nilai f(2) adalah ...
f(2) = 2(2²) + 3(2) - 5 = 2(4) + 6 - 5 = 8 + 6 - 5 = 9 → **Jawaban: A**
2. Jika g(x) = x³ - 2x² + x + 4, maka nilai g(-1) adalah ...
g(-1) = (-1)³ - 2(-1)² + (-1) + 4 = -1 - 2(1) - 1 + 4 = -1 - 2 - 1 + 4 = 0 → **Jawaban: C**
3. Jika f(x) = ax² + 2x + 1 dan f(1) = 8, maka nilai a adalah ...
f(1) = a(1)² + 2(1) + 1 = a + 2 + 1 = a + 3. Diberi f(1) = 8 → a + 3 = 8 → a = 8 - 3 = 5 → **Jawaban: B**
4. Jika g(x) = 3x² + ax - 1 dan g(1) = 8, maka nilai a adalah ...
g(1) = 3(1)² + a(1) - 1 = 3 + a - 1 = a + 2. Diberi g(1) = 8 → a + 2 = 8 → a = 8 - 2 = 6 → **Jawaban: D**
5. Volume suatu balok (dalam cm³) dapat dimodelkan oleh fungsi V(x) = 2x³ - 3x² + 4x + 5, dengan x adalah panjang rusuk dalam cm. Tentukan volume balok jika panjang rusuknya 2 cm.
V(2) = 2(2³) - 3(2²) + 4(2) + 5 = 2(8) - 3(4) + 8 + 5 = 16 - 12 + 8 + 5 = 17 → **Jawaban: C**
6. Sebuah perusahaan memodelkan keuntungan bersih (dalam juta rupiah) dengan fungsi P(x) = -x³ + 6x² - 5x + 12, dengan x adalah jumlah produk terjual (dalam ribuan unit). Jika terjual 4 ribu unit, berapa keuntungan bersihnya?
P(4) = -(4³) + 6(4²) - 5(4) + 12 = -64 + 6(16) - 20 + 12 = -64 + 96 - 20 + 12 = 24 → **Jawaban: C**

4. Pembagian Polinomial

Dua pendekatan umum:

a. Metode Pembagian Bersusun

Teknik analog dengan pembagian bilangan: bagi suku tertinggi, kalikan, kurangi, turunkan suku berikutnya, ulangi sampai selesai. Hasilnya berupa hasil bagi dan sisa.

Contoh singkat: (x³ − 2x² + 4) ÷ (x − 1) → lakukan langkah bersusun untuk mendapatkan kuosien dan sisa.

i. Metode Horner

Horner menyederhanakan pembagian oleh (x − c) sekaligus mengevaluasi P(c). Susun koefisien, turunkan langkah perkalian & penjumlahan berulang untuk memperoleh kuosien dan sisa.

ii. Metode Horner (variasi/quick)

Variasi penulisan Horner yang lebih ringkas, kadang disebut quick Horner, digunakan untuk perhitungan cepat tanpa menulis variabel secara eksplisit.

1. Tentukan hasil bagi dan sisa dari pembagian 2x³ + 3x² - 2x + 5 oleh x - 2 menggunakan metode Horner.
Gunakan metode Horner dengan pembagi (x - 2), k = 2. Susun: 2 | 3 | -2 | 5 Turunkan 2 → kalikan 2×2=4 → jumlah 3+4=7 → kalikan 7×2=14 → jumlah -2+14=12 → kalikan 12×2=24 → jumlah 5+24=29. Hasil bagi: 2x² + 7x + 12, sisa 29. → **Jawaban: A**
2. Tentukan hasil bagi dan sisa dari pembagian x³ + 4x² - 5x - 14 oleh x + 3 menggunakan metode Horner.
k = -3 (dari x + 3). Susun: 1 | 4 | -5 | -14 Turunkan 1 → kalikan -3 → 1×-3=-3 → jumlah 4-3=1 → kalikan -3 → 1×-3=-3 → jumlah -5-3=-8 → kalikan -3 → -8×-3=24 → jumlah -14+24=10. Hasil bagi: x² + x - 8, sisa 10. → **Jawaban: B**
3. Tentukan hasil bagi dan sisa dari pembagian 2x³ - 5x² - 4x + 9 oleh x - 4 menggunakan metode susun.
Metode susun: Bagi 2x³ : x = 2x² → kalikan (2x²)(x - 4) = 2x³ - 8x² → kurangi: -5x² - (-8x²) = 3x². Turunkan -4x → bagi 3x² : x = 3x → kalikan (3x)(x - 4) = 3x² - 12x → kurangi: -4x - (-12x) = 8x. Turunkan +9 → bagi 8x : x = 8 → kalikan (8)(x - 4) = 8x - 32 → kurangi: 9 - (-32) = 41. Hasil bagi: 2x² + 3x + 8, sisa 41. → **Jawaban: A**
4. Tentukan hasil bagi dan sisa dari pembagian x³ - 3x² + 4x - 12 oleh x + 1 menggunakan metode susun.
Metode susun: Bagi x³ : x = x² → kalikan (x²)(x + 1) = x³ + x² → kurangi: -3x² - x² = -4x². Turunkan +4x → bagi -4x² : x = -4x → kalikan (-4x)(x + 1) = -4x² - 4x → kurangi: 4x - (-4x) = 8x. Turunkan -12 → bagi 8x : x = 8 → kalikan (8)(x + 1) = 8x + 8 → kurangi: -12 - 8 = -20. Hasil bagi: x² - 4x + 8, sisa -20. → **Jawaban: A**

5. Teorema Faktor

Jika P(c) = 0, maka (x − c) adalah faktor dari P(x). Artinya, bila suatu nilai c membuat polinomial sama dengan nol, maka (x − c) membagi habis P(x).

Contoh Soal:

Tentukan apakah (x - 2) merupakan faktor dari polinomial P(x) = x³ - 4x² + 5x - 2.

Pembahasan:

Untuk menentukan apakah (x - 2) adalah faktor dari P(x), kita substitusi c = 2 ke dalam P(x):

P(2) = 2³ - 4(2)² + 5(2) - 2 = 8 - 16 + 10 - 2 = 0

Karena P(2) = 0, maka (x - 2) adalah faktor dari P(x).

6. Teorema Sisa

Sisa pembagian P(x) oleh (x − c) sama dengan P(c). Jadi, untuk mengetahui sisa pembagian, cukup evaluasi polinomial P pada x = c.

Contoh Soal:

Hitung sisa pembagian polinomial P(x) = 2x³ - 3x² + 4x - 5 oleh (x - 1).

Pembahasan:

Untuk mencari sisa pembagian, kita evaluasi P(x) pada x = 1:

P(1) = 2(1)³ - 3(1)² + 4(1) - 5 = 2 - 3 + 4 - 5 = -2

Jadi, sisa pembagian P(x) oleh (x - 1) adalah -2.

6b. Contoh Soal Teorema Sisa

Diketahui polinomial f(x) memiliki sifat berikut:

  • Sisa pembagian f(x) oleh (x - 2) adalah 4
  • Sisa pembagian f(x) oleh (x + 2) adalah -8

Pertanyaan:

Tentukan sisa pembagian f(x) oleh (x² - 4).

Pembahasan:

Karena \((x^2 - 4) = (x - 2)(x + 2)\), sisa pembagian f(x) oleh \((x^2 - 4)\)adalah polinomial berderajat kurang dari 2, yaitu berbentuk:

\[ R(x) = ax + b \]

Dengan menggunakan teorema sisa, kita substitusi nilai \(x = 2\) dan \(x = -2\):

  • \(R(2) = 2a + b = f(2) = 4\)
  • \(R(-2) = -2a + b = f(-2) = -8\)

Langkah 1: Bentuk sistem persamaan

\[ \begin{cases} 2a + b = 4 \\ -2a + b = -8 \end{cases} \]

Langkah 2: Cari nilai \(a\) dan \(b\)

Kurangkan persamaan kedua dari yang pertama:

\[ (2a + b) - (-2a + b) = 4 - (-8) \implies 4a = 12 \implies a = 3 \]

Substitusikan \(a = 3\) ke persamaan pertama:

\[ 2(3) + b = 4 \implies 6 + b = 4 \implies b = 4 - 6 = -2 \]

Kesimpulan:

Jadi, sisa pembagian f(x) oleh \((x^2 - 4)\)adalah:

\[ R(x) = 3x - 2 \]

7. Metode Partial

Metode Partial digunakan untuk memecah fungsi rasional menjadi pecahan sederhana (partial fractions), berguna dalam integral dan analisis aljabar.

Contoh Soal:

\[ \frac{2x + 3}{x^2 - 1} = \frac{A}{x - 1} + \frac{B}{x + 1} \]

Cari nilai A dan B sehingga persamaan tersebut benar.

Langkah 1: Faktorkan penyebut

\( x^2 - 1 = (x - 1)(x + 1) \), sehingga kita pisahkan menjadi dua pecahan sederhana.

Langkah 2: Kalikan kedua ruas dengan penyebut bersama

\[ 2x + 3 = A(x + 1) + B(x - 1) \]

Langkah 3: Cari nilai A dan B dengan substitusi nilai x

  • Untuk x = 1:

    2(1) + 3 = A(1 + 1) + B(0) → 5 = 2A → A = 2.5

  • Untuk x = -1:

    2(-1) + 3 = A(0) + B(-1 - 1) → 1 = -2B → B = -0.5

Kesimpulan:

Jadi, \[ \frac{2x + 3}{x^2 - 1} = \frac{2.5}{x - 1} - \frac{0.5}{x + 1} \]

7b. Metode Partial: Penyebut (x - 1)(x^2 + 2)

Jika penyebut merupakan hasil perkalian faktor linear dan kuadrat irreducible, bentuk partial fraction adalah kombinasi pecahan dengan penyebut masing-masing faktor.

Contoh Soal:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x^2 + 2)} = \frac{A}{x - 1} + \frac{Bx + C}{x^2 + 2} \]

Tentukan nilai A, B, dan C.

Langkah 1: Bentuk partial fraction

Bentuk pecahan parsial adalah: \[ \frac{A}{x - 1} + \frac{Bx + C}{x^2 + 2} \]

Langkah 2: Kalikan kedua ruas dengan penyebut

\[ 3x^2 + 5x + 2 = A(x^2 + 2) + (Bx + C)(x - 1) \]

Langkah 3: Kembangkan ruas kanan

\[ 3x^2 + 5x + 2 = A x^2 + 2A + B x^2 - B x + C x - C \]

Langkah 4: Gabungkan suku sejenis

\[ 3x^2 + 5x + 2 = (A + B) x^2 + (-B + C) x + (2A - C) \]

Langkah 5: Samakan koefisien

  • Koefisien \(x^2\): \(3 = A + B\)
  • Koefisien \(x\): \(5 = -B + C\)
  • Konstanta: \(2 = 2A - C\)

Langkah 6: Selesaikan sistem persamaan

Dari persamaan di atas:

\[ \begin{cases} 3 = A + B \\ 5 = -B + C \\ 2 = 2A - C \end{cases} \]

Dengan substitusi:

  • \(B = 3 - A\)
  • \(5 = -(3 - A) + C \Rightarrow C = 8 - A\)

  • \(2 = 2A - (8 - A) \Rightarrow 3A = 10 \Rightarrow A = \frac{10}{3}\)

  • \(B = 3 - \frac{10}{3} = -\frac{1}{3}\)
  • \(C = 8 - \frac{10}{3} = \frac{14}{3}\)

Kesimpulan:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x^2 + 2)} = \frac{\frac{10}{3}}{x - 1} + \frac{-\frac{1}{3}x + \frac{14}{3}}{x^2 + 2} \]

7c. Metode Partial: Penyebut dengan Pangkat

Jika penyebut mengandung faktor pangkat, misal \((x - 1)(x + 1)^2\), bentuk partial fractions harus mencakup pecahan untuk setiap pangkat tersebut.

Contoh Soal:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x + 1)^2} = \frac{A}{x - 1} + \frac{B}{x + 1} + \frac{C}{(x + 1)^2} \]

Tentukan nilai A, B, dan C.

Langkah 1: Bentuk partial fraction lengkap

Karena ada faktor \((x+1)^2\), kita buat dua pecahan: satu untuk pangkat 1 dan satu untuk pangkat 2.

Langkah 2: Kalikan kedua ruas dengan penyebut

\[ 3x^2 + 5x + 2 = A(x + 1)^2 + B(x - 1)(x + 1) + C(x - 1) \]

Langkah 3: Substitusi nilai x untuk cari A, B, C

  • Untuk x = 1:
    \(3(1)^2 + 5(1) + 2 = A(1 + 1)^2 + B(0) + C(0)\) → \(3 + 5 + 2 = A \times 4\) → \(10 = 4A\) → \(A = \frac{10}{4} = 2.5\)
  • Untuk x = -1:
    \(3(-1)^2 + 5(-1) + 2 = A(0) + B(-1 - 1)(0) + C(-1 - 1)\) → \(3 - 5 + 2 = C \times (-2)\) → \(0 = -2C\) → \(C = 0\)

Langkah 4: Cari B dengan substitusi nilai x lain

Pilih nilai lain, misal x = 0:

\(3(0)^2 + 5(0) + 2 = A(0 + 1)^2 + B(0 - 1)(0 + 1) + C(0 - 1)\)
\(2 = A(1)^2 + B(-1)(1) + C(-1)\)
\(2 = A - B - C\)

Dengan \(A = 2.5\) dan \(C = 0\), maka:

\[ 2 = 2.5 - B - 0 \implies B = 2.5 - 2 = 0.5 \]

Kesimpulan:

\[ \frac{3x^2 + 5x + 2}{(x - 1)(x + 1)^2} = \frac{2.5}{x - 1} + \frac{0.5}{x + 1} + \frac{0}{(x + 1)^2} = \frac{2.5}{x - 1} + \frac{0.5}{x + 1} \]

Karena \(C = 0\), suku terakhir hilang.
